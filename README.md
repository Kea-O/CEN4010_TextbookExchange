# CEN4010 Group Project - *Textbook Exchange*

Submitted by: **Keagan O'Leary, Caden Bird, Julian Arnau, Matthew Malayil, David Nguyen**

**Textbook Exchange** is a textbook exchange app created for Principles of Software Engineering, CEN4010, at Florida Atlantic University. This app allows users to post textbooks they're willing to sell, with a messaging system letting buyers inquire or haggle with the sellers. Buyers can browse a post feed or search for specific textbooks.

Time spent: **43** hours spent in total

## Features

The following functionality is completed:

- [X] Users can signup or login to the app
- [X] Users can post textbooks they have for sale. This includes an image, price, title, author, isbn, edition, condition, subject, meetup times and meetup locations.
- [X] Users can message posters selling textbooks
- [X] The app gives a notification when a message is received
- [X] Users messaging each other can review each other out of 5 stars for how cooperative the other was during the sale
- [X] User's scores are shown in their posts so that other users know what to expect when messaging them
- [X] Users can delete old posts after they've sold the textbook
- [X] Users can see what they've posted in its own tab, and their own posts don't show up for them when browsing/searching
